# Contributing to ImmichKiosk

## Getting Started
To get started check out the contributing guide on [the docs site](https://docs.immichkiosk.app/contributing/).

## General
Please try to keep pull requests as focused as possible. A PR should aim to do one thing. The smaller a PR, the easier it will be to review, the fewer changes are likely needed, and the quicker it will likely be merged.

## Use of generative AI
Please avoid using generative AI to write code for this project. Reviewing PRs generated by a LLM can be very inefficient and frustrating and the PR is likely to be rejected.
